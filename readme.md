## 👇 Clone the repository
- Open your terminal to write theses lines:
```bash
  ## clone repository
  $ git clone https://github.com/danilobandeira29/template-backend.git

  ## access project directory
  $ cd template-backend

  ## install all dependencies (or you can use 'npm install')
  $ yarn

```
